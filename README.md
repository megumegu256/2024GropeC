## 概要
基本マウス操作で銃の照準を合わせ、目の前に次々出現する的をクリックで撃ってスコアやを競うゲーム

## ルール
マウスカーソルを動かし、的に合わせた状態で左クリックすることで倒すことができる

### 的の種類
- スコア5点 - 小さいサイズの敵
- スコア3点 - 普通のサイズの敵
- スコア1点 - 大きいサイズの敵
